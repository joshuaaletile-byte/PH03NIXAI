# PHO3NIXAI Project Structure

## Main Systems

PHO3NIXAI is divided into independent systems that work together.

---

# 1. AI Core

Folder:
Purpose:
The brain of PHO3NIXAI.

Contains:
- AI communication
- Command interpretation
- Memory handling
- Personality settings
- Response generation

---

# 2. Voice System

Folder:
Purpose:
Handles communication through speech.

Contains:
- Speech recognition
- Text-to-speech
- Wake word system
- Voice commands

---

# 3. User Interface

Folder:
Purpose:
The PHO3NIXAI visual experience.

Contains:
- Home screen
- Chat interface
- HUD animation
- Settings screen
- Assistant visualizer

---

# 4. Device Control

Folder:
Purpose:
Controls supported phone actions.

Contains:
- App launcher
- System shortcuts
- Flashlight control
- Device actions

---

# 5. File Intelligence

Folder:
Purpose:
Manages user-approved files.

Contains:
- File search

- File information

---

# 6. Productivity

Folder:
Purpose:
Helps manage daily tasks.

Contains:
- Alarms
- Timers

- Calendar

# 7. Security

Folder:
Purpose:

Folder:
